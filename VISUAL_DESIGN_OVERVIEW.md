## Responsive Behavior

### Desktop (1024px+)
- Full 2-column layout with sidebar
- Grid cards: 2-4 columns
- All elements visible

### Tablet (768px-1023px)
- Sidebar visible but narrower
- Grid cards: 1-2 columns
- Adjusted spacing

### Mobile (<768px)
- Sidebar hidden, hamburger menu
- Single column layout
- Full-width modals
- Larger touch targets

- Dark text

- Dark blue backgrounds (#1e293b)
- Light text
- Dark borders
- Slightly adjusted shadows
